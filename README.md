Leishmania donovani

Вид жгутиконосных паразитических протистов рода Leishmania, является возбудителем кала-азара — антропонозного висцерального лейшманиоза. Единственным доказанным естественным резервуаром является человек. Переносчиком служат несколько видов москитов рода Phlebotomus.

Упоминается в 13 публикациях на PubMed. Размер генома 32.4 Мб, N50 равен 1 Мб. Включает 8123 гена, из них 7953 кодируют белки.
